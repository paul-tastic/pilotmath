Pilot Math!

Project Summary

After each flight, pilots need to fill out all sorts of forms, and these forms require flight times in minutes and tenths of hours. Although might not seem difficult, after a hectic flight, that math can be tricky.

I created Pilot Math to help me (and other pilots) get that math correct with minimal pain.

It was originally written in Swift for iPhone/iPad deployment, and then I adapted it to javascript for use in a web browser. I also wrote a jQuery version to demonstrate proficiency with that javascript library.
